# Neo-Mesh Suggested Configs

Suggested node configurations for various roles in the Neo-Mesh network.
These configs are community-maintained starting points — read through them before deploying a device on either network.

---

## Contents

### [MeshCore Repeater Config](./meshcoreRepeaterConfig.md)
Suggested settings for deploying a **MeshCore repeater** node.
Use this if you are running MeshCore firmware on any supported device and want to contribute infrastructure to the mesh.

### [Meshtastic Config](./meshtasticConfig.md)
Suggested settings for deploying a **Meshtastic** node.
Use this if you are running Meshtastic firmware and want to ensure your node is configured consistently with the rest of the Neo-Mesh network.

---

## Which Should I Use?

| I'm running... | Go to... |
|---|---|
| MeshCore firmware | [meshcoreRepeaterConfig.md](./meshcoreRepeaterConfig.md) |
| Meshtastic firmware | [meshtasticConfig.md](./meshtasticConfig.md) |

> Not sure which firmware to use? MeshCore and Meshtastic are separate protocols and are **not compatible with each other**. Check the Neo-Mesh Discord for guidance on which the network to use.

---

## General Reminders

- Always connect your antenna before powering on
- Repeater and router roles should only be used for well-placed, elevated nodes.
- Coordinate with the community before deploying infrastructure nodes
- Keep your firmware up to date
